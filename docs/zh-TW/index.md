# 星辰筆記 (Stellar Note)

!!! quote "破碎天空之下"
    數千年來，這片大陸的天空未曾有過月亮與星光。
    月亮早已碎裂，化作蒼穹的永久傷痕；星辰亦已隱沒，成為吟遊詩人口中的遙遠神話。

## 第一章：引言

如今，我們所知的一切，全是在廢墟上重建而來。

在這短晝的世界裡指引方向的，是來自南方森林的精靈女王 **伊爾瓦納 (Il’wana)**。她強大的法力有如一座永不熄滅的燈塔，其光芒所及之處，便被稱為「已知世界」。

但在光芒的陰影下，權力的遊戲未曾停歇：
* **北方獸人**南遷留下的傷痕尚未痊癒。
* **選帝侯帝國**因彼此猜忌和對權力的欲望四分五裂。
* **南方王國**在精靈扶植下崛起，對土地的渴望化為暗流。
* **精靈巡禮者**在世間遊走，有意無意地影響勢力平衡。

這是一個秩序逐漸崩潰、英雄與惡棍的邊界晦暗不明的時代。被逼入絕境者、野心家、殘存的忠臣，無不伸長雙手，企圖從遮蔽星空的月之塵中，抓住自己的命運。

> 而你，一位應運而生的指揮官，將集結手下，運用戰術與勇氣，在這破碎的天空之下，為你的奮起、你的君主，或你的野心踏上沙場。
