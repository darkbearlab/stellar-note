# 進階規則 (Advanced Rules)

!!! warning "異數規則"
    這些規則將賦予你更強大的力量，也將帶來更艱鉅的挑戰。
## 擴充規模
進階遊戲建議規模為 **3+2**。追加的點數可用於購買英雄或換成 **6點升級點**。

## 訓練 (Training)
升級點數可用於讓單位獲得「訓練」。

### 一般訓練 (1點)
* **精準射擊**：首次射擊 +1 命中強度。
* **無視苦痛**：續戰力檢定視為 +1。
* **威脅預感**：成為注視目標時，迴避 +1。

### 英雄訓練 (2點)
* **榮耀衝鋒**：移動後接戰可免費進行強力近戰。
* **騎士導師**：隊伍加入一名「侍從」。

## 坐騎 (Mounts)
單位可消耗升級點獲得坐騎 (普通馬、戰馬、覆甲戰馬)。

* **模型**：佔據相鄰兩格 (2格)。
* **移動**：像貪食蛇一樣移動，後方格跟隨前方格。
* **落馬檢定**：被命中或馬匹死亡時需檢定，失敗則倒地並受傷。
