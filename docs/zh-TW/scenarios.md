# 戰場劇本

## 5.1 寸土不讓 (Contested Ground)
> [cite_start]「沒有撤退的餘裕...唯一的選擇只有踏過敵人的屍體。」[cite: 352]

* **戰場**：12x14 格。
* **勝利**：5 回合後分數高者勝。
* [cite_start]**得分**：擊敗英雄 (3分)、士兵 (1分)、每三個追隨者 (1分)。[cite: 361]

## 5.2 唯一的井 (The Only Well)
> [cite_start]「海市蜃樓的另一端，究竟是甘美的湧泉，還是染血的刀劍。」[cite: 366]

* [cite_start]**特殊規則 [乾渴]**：所有單位初始續戰 -1。[cite: 372]
* [cite_start]**勝利**：控制「真正的古井」得分。[cite: 379]

## 5.3 絕望隘口 (Desperate Pass)
* **非對稱戰鬥**：伏擊方 vs 突圍方。
* [cite_start]**勝利**：突圍方需讓半數以上單位逃離戰場。[cite: 395]
