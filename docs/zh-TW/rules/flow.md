# 第三章：遊戲流程

## 3.1 流程概覽
1.  雙方決定情境與部署。
2.  遊戲開始，依回合順序進行。

## 3.2 回合詳細流程

### 1. 續戰力階段 (Morale Phase)
[cite_start]所有累積了「命中強度」的單位進行檢定。[cite: 125]
* [cite_start]**檢定方式**：單位擲 `[續戰]` 骰 vs 對手擲 `[命中強度]` 骰。[cite: 128]
* **成功**：繼續戰鬥。
* **失敗**：
    * [cite_start]若有多條血量 (如 4/2/1)，移除當前血條，命中歸零，使用下一階數值。[cite: 131]
    * [cite_start]若無下一階數值，單位**被擊敗** (移出戰場)。[cite: 135]

### 2. 恢復階段 (Recovery Phase)
[cite_start]所有我方單位完整恢復 `[行動點]`。[cite: 138]

### 3. 啟動階段 (Activation Phase)
玩家輪流啟動單位。
* [cite_start]**啟動開始**：移除迴避標記。[cite: 145]
* **行動**：移動、攻擊、施法等。
* [cite_start]**啟動結束**：若移動距離大於 2 格，獲得「迴避標記」。[cite: 147]

### 4. 結束階段 (End Phase)
[cite_start]檢查勝利條件。若未達成則換下一位玩家。[cite: 151]
