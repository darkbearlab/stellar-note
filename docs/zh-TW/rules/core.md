# 第二章：遊戲核心

## 2.1 遊戲準備
* **遊戲地圖**：六邊形方格地圖。
* **骰子**：六面骰 (D6)，建議準備多顆。

## 2.2 擲骰與拚骰 (Opposed Rolls)
遊戲透過**拚骰**來決定行動成功與否。

!!! info "拚骰機制"
    1. 雙方投擲指定數量的骰子。
    2. **比大小**：比較雙方「最大」的那顆骰子。較大者獲勝。
    3. **計算得分**：獲勝方「剩餘的骰子數」即為得分。失敗方得 0 分。
    4. **平手處理**：若最大值相同，各自移除該骰子，比較下一顆。若骰子耗盡仍平手，則雙方 0 分。

> **例：** 玩家A (5, 5) vs 玩家B (5, 4, 2)。
> 第一輪 5 比 5 (平手移除) -> 第二輪 A是5，B是4 (A勝)。A剩餘 1 顆骰子，得 1 分。

## 2.3 單位資訊
所有單位都有以下素質：

| 屬性 | 描述 |
| :--- | :--- |
| **迴避** | 被非近戰攻擊時的防禦骰數。 |
| **裝甲** | 減少命中強度 (傷害減免)。 |
| **續戰** | 生命力與士氣。格式如 `4/2/1` 代表多條血量。 |
| **行動點** | 用於支付行動的資源 (AP)。 |

## 行動規則

### 移動 (Movement)
* **消耗**：依據單位資料。
* **規則**：移動到鄰近一格。若在敵方 1 格內，消耗 +1。
* **地形**：困難地形消耗 +1。

### 近戰與射擊 (Combat)
攻擊方投擲 `[骰數]`，防禦方投擲 `[迴避]`。

* **攻擊方加成**：
    * 目標是注視目標：+1 骰
    * 夾擊 (目標1格內有友軍)：每有一名 +1 骰
    * 近戰且目標倒地：+2 骰
    * 高地優勢：+1 骰
* **防禦方加成**：
    * 掩體：+1 骰 
    * 距離 (每3格)：+1 骰 
    * 高地優勢：+1 骰 

**傷害計算：**
`拚骰得分 + 行動[威力] - [裝甲] = 累積命中強度`

### 施法 (Magic)
施法者與目標進行拚骰。
* **破法**：目標 6 格內的對手可消耗「破法標記」來增加防禦骰數。 
* **施法骰**：依據法術骰數。 
* **抵抗骰**：`1顆基礎骰 + 破法標記數`。 

## 戰場狀態
* **視線 (LoS)**：兩格中心連線，若經過粗實線則遮蔽。
* **掩體**：視線經過粗虛線。
* **注視目標**：英雄可標記一名敵人。
* **倒地**：防禦力下降，失去注視目標，僅能移動。
