# 第二章：遊戲核心

## 2.1 遊戲準備
* [cite_start]**遊戲地圖**：六邊形方格地圖。[cite: 14]
* [cite_start]**骰子**：六面骰 (D6)，建議準備多顆。[cite: 17]

## 2.2 擲骰與拚骰 (Opposed Rolls)
遊戲透過**拚骰**來決定行動成功與否。

!!! info "拚骰機制"
    1. [cite_start]雙方投擲指定數量的骰子。[cite: 22]
    2. [cite_start]**比大小**：比較雙方「最大」的那顆骰子。較大者獲勝。[cite: 23]
    3. [cite_start]**計算得分**：獲勝方「剩餘的骰子數」即為得分。失敗方得 0 分。[cite: 24]
    4. [cite_start]**平手處理**：若最大值相同，各自移除該骰子，比較下一顆。若骰子耗盡仍平手，則雙方 0 分。[cite: 25, 27]

> **例：** 玩家A (5, 5) vs 玩家B (5, 4, 2)。
> [cite_start]第一輪 5 比 5 (平手移除) -> 第二輪 A是5，B是4 (A勝)。A剩餘 1 顆骰子，得 1 分。[cite: 30]

## 2.3 單位資訊
[cite_start]所有單位都有以下素質：[cite: 36]

| 屬性 | 描述 |
| :--- | :--- |
| **迴避** | [cite_start]被非近戰攻擊時的防禦骰數。[cite: 39] |
| **裝甲** | [cite_start]減少命中強度 (傷害減免)。[cite: 40] |
| **續戰** | [cite_start]生命力與士氣。格式如 `4/2/1` 代表多條血量。[cite: 41] |

## 行動規則

### 移動 (Movement)
* **消耗**：依據單位資料。
* [cite_start]**規則**：移動到鄰近一格。若在敵方 1 格內，消耗 +1。[cite: 49, 52]
* **地形**：困難地形消耗 +1。

### 近戰與射擊 (Combat)
攻擊方投擲 `[骰數]`，防禦方投擲 `[迴避]`。

* **攻擊方加成**：
    * [cite_start]目標是注視目標：+1 骰 [cite: 59]
    * [cite_start]夾擊 (目標1格內有友軍)：每有一名 +1 骰 [cite: 60]
    * [cite_start]近戰且目標倒地：+2 骰 [cite: 61]
    * [cite_start]高地優勢：+1 骰 [cite: 62]
* **防禦方加成**：
    * [cite_start]掩體：+1 骰 [cite: 64]
    * [cite_start]距離 (每3格)：+1 骰 [cite: 66]
    * [cite_start]高地優勢：+1 骰 [cite: 67]

**傷害計算：**
[cite_start]`拚骰得分 + 行動[威力] - [裝甲] = 累積命中強度` [cite: 71]

### 施法 (Magic)
施法者與目標進行拚骰。
* [cite_start]**破法**：目標 6 格內的對手可消耗「破法標記」來增加防禦骰數。[cite: 94]
* **施法骰**：依據法術骰數。
* [cite_start]**抵抗骰**：`1顆基礎骰 + 破法標記數`。[cite: 95]

## 戰場狀態
* [cite_start]**視線 (LoS)**：兩格中心連線，若經過粗實線則遮蔽。[cite: 74, 76]
* [cite_start]**掩體**：視線經過粗虛線。[cite: 78]
* **注視目標**：英雄可標記一名敵人。
* [cite_start]**倒地**：防禦力下降，失去注視目標，僅能移動。[cite: 84-88]
